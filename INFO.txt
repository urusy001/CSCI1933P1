How to Compile and Run:
- From the parent directory (the directory containing the CSCI1933P1 folder):
  - Compile: javac CSCI1933P1/FractalProgram.java
  - Run: java CSCI1933P1.FractalProgram
- When prompted, enter one of: triangle, rectangle, or circle

Assumptions:
- User input is expected to be one of the three supported shape names: triangle, rectangle, or circle.
- Fractal recursion depth is fixed at 7 levels in FractalDrawer.
- Program uses the default Canvas size (800x800) created by new Canvas().
- Total area is the sum of all drawn shapes without subtracting overlap.

Additional Features Implemented:
- Input validation for missing/empty/invalid shape input in FractalProgram.
- Case-insensitive shape selection (triangle, rectangle, circle).
- Color variation across recursive calls using generated HSB colors.

Known Bugs / Defects:
- Circle.calculateArea() currently uses an incorrect formula and does not depend on radius, so circle area values (and total area for circle fractals) are incorrect.
- On invalid input, the program prints an error and exits instead of re-prompting the user.
